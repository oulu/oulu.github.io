last update2014/10/06

+ie-rgba( )

不透明度の値が css で使うことが出来ないレガシーなブラウザ用の mixin です。不透明度の値が css で使うことが出来ないレガシーなブラウザには、不透明度のある表示したい色と、その背景にある色をミックスして、擬似的な不透明度のある色を出力します。ただし、css プロパティが background-color の場合のみ、不透明度のある表示したい色と、その背景にある色をミックスした色を出力するのではなく、png の image ファイルを生成し、それを背景画像にあてた設定で出力されます。

引数

  • 第一引数に不透明度のある表示したい色の値を渡します。
  • 第二引数に第一引数に渡した色の背景にある色を渡します。
  • 第三引数に第一引数で渡した色を使うプロパティを渡します。

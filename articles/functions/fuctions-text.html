text_align( )

渡された引数が css プロパティ text-align で使用できる値か否かを判別するための function です。leftrightcenter のいずれかの場合、true を返します。それ以外は null を返します。

text_align(right)

$value: center
text_align($value)

上記の場合、text_align() に渡された引数は css プロパティ text-align で使用できる値なので、 true を返します。

grids

last update2014/10/06

oulu grid

oulu は、twitter bootstrap と同じルールの grid システムを持っています。

.row

.row という class を持つブロック内で grid　システムが有効になります。

.col

.row 内に .col というプレフィックスを持った class を持つブロックが grid　システムの一つの　grid　のブロックになります。記法は以下になります。

記法のルール

1
.col-{デバイスのサイズ}-{ブロックの幅}
